# Chronos-OSINT

**Инструмент для поиска пропавших людей и факт-чекинга журналистских расследований.**

С помощью компьютерного зрения и открытых геоданных Chronos-OSINT помогает восстановить географическую привязку снимка даже при отсутствии метаданных. Используется волонтёрами поисковых отрядов и независимыми расследователями.

## Возможности
- Извлечение GPS-координат из скрытых слоёв изображений (включая превью-тамбнейлы).
- Расчёт местоположения по тени и времени съёмки (уникальный алгоритм с учётом азимута и высоты солнца).
- Многоуровневый поиск по панорамам, спутниковым снимкам и соцсетям (Google, Yandex, Bing, PimEyes и др.).
- Автоматическая обработка капч и ротация прокси (включая Tor с сигналом NEWNYM).
