# Chronos-OSINT

**Инструмент для поиска пропавших людей и факт-чекинга журналистских расследований.**

С помощью компьютерного зрения и открытых геоданных Chronos-OSINT помогает восстановить географическую привязку снимка даже при отсутствии метаданных. Используется волонтёрами поисковых отрядов и независимыми расследователями.

## Возможности
- Извлечение GPS-координат из скрытых слоёв изображений (включая превью-тамбнейлы).
- Расчёт местоположения по тени и времени съёмки (уникальный алгоритм с учётом азимута и высоты солнца).
- Многоуровневый поиск по панорамам, спутниковым снимкам и соцсетям (Google, Yandex, Bing, PimEyes и др.).
- Автоматическая обработка капч и ротация прокси (включая Tor с сигналом NEWNYM).

## Установка
```bash
git clone https://github.com/yourusername/chronos-osint.git

## Использование

```bash
python main.py /path/to/photo.jpg
```

На выходе вы получите JSON с координатами, методом определения и уверенностью.

## Ограничения

· Модуль shadow_geo предполагает, что снимок сделан практически сверху (Nadir) или что известен масштаб изображения. Без калибровки камеры длина тени в пикселях не может быть напрямую переведена в метры, поэтому алгоритм использует только направление тени - это даёт грубое приближение. Для повышения точности предоставляйте эталонный объект известного размера.
· Для работы с Google, Yandex и другими сервисами могут потребоваться API-ключи; без них поиск будет ограничен.
